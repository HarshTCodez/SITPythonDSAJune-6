# Day 2 — Python Fundamentals & Logic Building

## Topics Covered

1. **Functions Basics**
   - Defining and calling functions.
   - Understanding `return` vs `print`.
   - Function parameters: default arguments and keyword arguments.

2. **Call Stack in Depth**
   - How function calls stack (LIFO) and unwind.
   - Tracing function executions.

3. **The `print()` Function in Depth**
   - Using the `sep` and `end` arguments for advanced formatting.

4. **Lists Basics**
   - Creating and interacting with Lists.
   - Shallow copy vs. Deep copy in Python.

5. **Pattern Printing**
   - Two right-angled triangles meeting at the base (`*   *` / `** **` / `*****`).

6. **Problem Solving**
   - **Happy Number (LeetCode 202):** Using fast and slow pointers (Floyd's Tortoise and Hare algorithm) to detect cycles.
