# 15-Day Python + DSA Training

Teaching material for a 15-day Python and Data Structures & Algorithms course
for beginners. Each day has notes (logic + explanations) and runnable Python code.

## Structure

```
.
└── course/
    ├── README.md          # day-wise index
    ├── day_01/
    │   ├── day_01.md      # notes: logic + code for the day
    │   └── *.py           # runnable Python files
    └── day_02/
        └── day_02.md      # planned topics
```

## How to use

- Read the day's markdown file for the logic and explanations.
- Run any Python file directly:

  ```bash
  python3 course/day_01/factorial.py
  ```
